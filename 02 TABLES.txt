1. Tables consists of rows and coloumns similar to excel spreadsheet

2. To create a Table use the statement
CREATE TABLE nameofthetable (
    list the coloumns
    each coloumn is separated with a comma
);

3. After each coloumn we need to state the datatype of what will be storing

example:CREATE TABLE employees(
employee_id INT,
first_name VARCHAR(50),
last_name VARCHAR(50),
hourly_pay DECIMAL(5,2),
hire_date DATE
);

4. To select a Table use statement
SELECT * FROM nameofthetable;

5. To rename a Table use statement
RENAME TABLE previousname TO newname;

6. To drop a Table
DROP TABLE nameoftable;
